Privacy Policy – MacroLens

Last Updated: May 18, 2025

1. Data Controller

For privacy inquiries, contact: macrolenssp@gmail.com.

2. Data Collection

  • No personal data (name, email, age, gender, location) is collected or stored on external servers.
  • All user-provided information (age, weight, height, goals, meal photos) is stored locally on the device using AsyncStorage.
  • Future planned use of Firebase for anonymous usage metrics; no data linked to personal identity.

3. Purpose of Processing

Local data is used solely to:

  • Create a personalized nutritional profile.
  • Calculate caloric and macronutrient requirements.
  • Customize the in-app experience (charts, comparisons).

4. Legal Basis

The processing of data is based on the user’s explicit consent by accepting this Privacy Policy and using the application.

5. Sharing and Transfers

  • The only external communication is with the OpenAI API for image or dish name processing.
  • Data transmissions may occur to servers located outside Argentina.
  • No data is sold or shared with third parties.

6. Data Retention

All data remains stored locally on the device. Deleting the account or uninstalling the app removes all data.

7. User Rights

  • Access stored data via the 'Profile' section.
  • Delete data using the 'Delete Account' option.
  • Edit data through the 'Edit Profile' option.

8. Security Measures

Data is stored locally using standard React Native AsyncStorage practices.
No personal credentials are sent to proprietary servers.
Future Firebase Analytics usage will collect anonymous and secure usage data.

9. Minors

The application is not intended for users under 13 years of age. We do not knowingly collect data from minors.

10. Policy Updates

The policy version and date are available in Settings → Privacy Policy.
Significant changes will be notified within the app upon launch.

By using MacroLens, you agree to this Privacy Policy.
