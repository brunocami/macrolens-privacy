2. Data Collection

  • No personal data (name, email, gender, location) is collected or stored on external servers.
  • All user-provided information (age, weight, height, goals, meal photos, profile data) is stored locally on the device using AsyncStorage.

3. Purpose of Processing

Local data is used solely to:

  • Create and maintain a local nutritional profile.
  • Calculate caloric and macronutrient recommendations.
  • Customize the in-app experience (charts, comparisons).

4. Legal Basis

The processing of data is based on the user’s explicit consent by accepting this Privacy Policy and using the application.

5. Sharing and Transfers

  • The only external communication is for AI analysis (e.g. OpenAI API) of meal images or descriptions.
  • Data transmissions may occur to servers located outside Argentina for that purpose only.
  • No data is sold or shared with any other third parties.

6. User Profile

No registration or online account is required. You create a local profile by entering your age, weight, height, goals, and meal data. All profile data is stored only on your device via AsyncStorage. The in-app “Delete Profile” feature permanently erases all your local data, equivalent to uninstalling the App.

7. User Rights

  • Access stored data via the ‘Profile’ section in the App.
  • Delete data using the ‘Delete Profile’ option, which erases all local data.
  • Edit data through the ‘Edit Profile’ option.

8. Security Measures

Data is stored locally using standard React Native AsyncStorage practices.
No personal credentials are sent to proprietary servers for storage.

Parents or guardians assume full responsibility for their minor’s use of the App.

10. Data Retention & Deletion

All data remains stored locally on the device. Using the ‘Delete Profile’ option or uninstalling the App will remove all data permanently.
